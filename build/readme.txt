The Slack Bot plugin is a simple Slack notification plugin built to help alert WordPress site owners.

== Description ==

The Slack Bot plugin is a simple notification plugin built to help alert WordPress site owners when posts have been created or published. It sends a simple notification message to the specified slack channel the user requires it to.

At a glance, this plugin helps you:

* Configure your messages to be sent to slack.
* Configure which slack channel to receive your messages.
* Configure the logo to be used by your slack bot.

== Frequently Asked Questions ==

= What is my Slack Username? =

Your Slack username is your default account username on Slack. It usually is a combination of your first name and last name.

= What is my Slack WebHook? =

Your Slack webhook is simply a unique URL of yours that is generated by the Slack API. To generate yours, visit the Slack API page.
